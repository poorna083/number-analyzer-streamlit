# 🔢 Streamlit Number Lab

A simple and interactive **Streamlit web app** to perform multiple number-based operations and checks in one place.

---

## 🚀 Features

This app allows users to input a number and perform the following operations:

* ➕ Sum of digits
* ✖️ Factorial
* 🔄 Reverse a number
* 🔁 Palindrome check
* 🕵️ Spy number
* 💯 Perfect number
* 💪 Armstrong number
* 🔢 Disarium number
* 🔥 Strong number
* 🌿 Xylem / Phloem number
* 🔲 Automorphic number
* 📊 Fibonacci series
* 😊 Happy number

---

## 🖥️ Demo Preview

Enter a number and click any button to instantly see results.

* Python 🐍

## 📦 Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/streamlit-number-lab.git
cd streamlit-number-lab
```

2. Install dependencies:

```bash
pip install streamlit
```

---

## ▶️ Run the App

```bash
streamlit run number_problems.py
```
